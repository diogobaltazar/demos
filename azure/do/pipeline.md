    + `<ado-repo-name>-ADB`
    + initializing a new `<ado-pipeline-build-df> = azure-pipelines.yml`
+ run build by clicking *Queue*, which will build the last repo commit
+ the build progress can be tracked in
    + logs
    + summary
    + tests
